To run and study the examples in this tutorial, install the following open-source tools:

✅ Bluespec Compiler (bsc)
🔗 GitHub: B-Lang-org/bsc

✅ Verilog Simulator (verilator)
🔗 Veripool: Verilator

✅ Alternative Simulator (iverilog) (slower for large designs)

✅ Waveform Viewer (gtkwave) (available in package managers)

✅ Tutorial Materials
🔗 GitHub: Bluespec Tutorial

✅ Bluespec Flute RISC-V CPU & WindSoC SoC
🔗 GitHub: Bluespec Flute

💻 Preferred Platforms: Ubuntu & Debian Linux (MacOS is also supported)
