

# **Installation Guide**  

This guide provides instructions to install the necessary tools for running and studying the examples in this tutorial. All tools used are open-source.  

## **1. Required Installations**  

### **Bluespec Compiler (`bsc`)**  
The Bluespec compiler (`bsc`) is required to compile BSV (Bluespec SystemVerilog) code.  
🔗 **Download and install:** [GitHub: B-Lang-org/bsc](https://github.com/B-Lang-org/bsc)  

### **Verilog Simulator (`verilator`)**  
Verilator is an open-source Verilog simulator optimized for high-performance simulations.  
🔗 **Download and install:** [Veripool: Verilator](https://www.veripool.org)  

### **Alternative Verilog Simulator (`iverilog`)**  
If you prefer, you can install **Icarus Verilog (`iverilog`)**, which is included in most package managers. Note that `iverilog` is slower for large designs but works fine for small examples.  
🔗 **Installation:** Available via standard package managers (`apt`, `brew`, `pacman`, etc.)  

### **Waveform Viewer (`gtkwave`)**  
GTKWave is a GUI tool for viewing VCD waveform files generated by Verilog simulations.  
🔗 **Installation:** Available in standard package managers (`apt-get install gtkwave`, `brew install gtkwave`)

---

### **Installation Commands for Linux (Ubuntu/Debian)**  

You can install most dependencies using the following commands:  

```sh
# Install Bluespec BSC (Refer to the GitHub repo for latest installation steps)
git clone https://github.com/B-Lang-org/bsc.git
cd bsc
make
sudo make install

# Install Verilator
sudo apt-get install verilator -y

# Install Icarus Verilog (optional)
sudo apt-get install iverilog -y

# Install GTKWave
sudo apt-get install gtkwave -y
```
